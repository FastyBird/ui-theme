# FastyBird web UI theme

FastyBird web UI theme is a design system built with Storybook and Vue.

#### Installation

```
yarn add @fastybird/web-ui-theme
```

Import the FastyBird UI main style sheet in your application:

```
@import "~@fastybird/web-ui-teheme/web-ui-theme.css";
```
